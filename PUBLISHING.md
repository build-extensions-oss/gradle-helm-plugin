# New version publication

This page describes how to publish a new version of plugin and all related artifacts.

Steps:

1. Create release with version line `a.b.c`.
   1. Inside the release - the tag with the same version name
2. Go to [Actions](https://github.com/build-extensions-oss/gradle-helm-plugin/actions) and approve deployment.

Please note - secrets are hidden under `Publishing` environment to avoid accidental usage.
